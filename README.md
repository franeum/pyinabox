# python in a box

micropython object in a box equipped with a esp32 microcontroller

## Connections (v0.1)

| **DEVICE**  | **pin** | **ESP32** |
| ----------- | ------- | --------- |
| PUSH Button | 1       | 32        |
| PUSH Button | 2       | GND       |
| POT         | V       | 3V3       |
| POT         | GND     | GND       |
| POT         | Signal  | 33        |
| SPEAKER     | +       | 25        |
| SPEAKER     | -       | GND       |
